# 祐天寺若麦（喵梦 / Amoris）角色分析

---

## PART A：角色记忆（客观事实与经历）

### 一、基本身份

| 属性 | 内容 |
|------|------|
| 本名 | 祐天寺若麦（ゆうてんじ にゃむ） |
| 通称 | 喵梦（にゃむ）、喵梦亲（にゃむちん） |
| 乐队角色 | Ave Mujica 鼓手，代号 **Amoris（爱恋骑士）** |
| 身份 | 高中生、美妆系YouTuber、综艺艺人、鼓手 |
| 学校 | 不明（非羽丘、非月之森） |
| 家庭 | 有母亲、哥哥、姐姐（在乡下老家），家庭并不富裕 |
| 生日 | 6月1日 |

> **名字含义**：母亲曾说——「喵梦的名字可是写作『年轻的麦子』呢。你会像麦子一样经历挫折而变得更强的。」

### 二、关键经历与时间线

#### 1. 加入 Ave Mujica 之前
- 从地方来到东京，带着「约法三章」——必须好好完成学业才得到家里同意
- 以「喵梦亲」名义在社交平台发布美妆视频和架子鼓演奏视频，积累了一定的粉丝
- 被祥子发掘，加入 Ave Mujica 担任鼓手（Amoris）

#### 2. Ave Mujica 假面时代
- 乐队以「假面舞会」世界观出道，成员均戴面具隐藏真容
- 出道后迅速走红，不到一个月登上武道馆
- 喵梦在视频中多次暗示自己是 Amoris，但热度始终上不去——「『喵梦』的热度完全上不去。在视频里放了那么多暗示，还怎么叫人大吃一惊」

#### 3. 武道馆摘假面事件（重大转折）
- **这是若麦主动发起的最关键行动**——她在武道馆演出中擅自摘下了所有人的假面
- 祥子质问：「为何要做出这种可耻的行径」
- 若麦的回应：
  > 「嗯，这不是大获成功嘛。你看，消息都没停过。在最棒的舞台、最棒的时机取下假面，除了今天哪还有别的机会。」
- 这一行为直接导致全员身份暴露：Oblivionis = 丰川祥子、Mortis = 若叶睦、Doloris = 三角初华（sumimi）、Timoris = 八幡海铃（DISRUPTION）

#### 4. 摘假面后的矛盾激化
- 若麦不满睦受到过多关注、自己成为「空气」：
  > 「风头全被睦子抢走，变成了空气。为了来看演出的人，我已经拼命挤出笑容接受了。」
- 她与祥子的理念冲突加剧——祥子坚持音乐性，若麦追求话题度和曝光率
- 在「Music Floor」电视节目事件中，若麦支持加入 Mortis 的即兴表演环节，认为「给予观众渴望的东西才是娱乐」
- 祥子反对故意中断演奏，两人激烈争吵

#### 5. 福冈公演与乐队解散
- Mortis（若叶睦的另一人格）声称「不会弹吉他」
- 若麦在彩排时爆发：
  > 「我忍你好久了……乐队和演出不是我们几个就能办成的，是无数人付出的心血，甚至不睡觉的努力才办成的。」
  > 「你会考虑到明天来看演出的人吗？不光门票钱，交通费、住宿费也要花很多钱。大家都期待Ave Mujica的演出。」
- 她宣布退出：「我没空和大小姐玩过家家。你说过只要有人退出Ave Mujica就解散对吧……什么把人生交给你。」
- 乐队在舞台上正式宣告解散

#### 6. 解散后的个人发展
- 乐队解散后一个月，若麦以个人身份活跃于综艺节目
- 在综艺中担任常驻嘉宾，被称为「喵梦亲」
- 她讲述自己的「另类生涯」——从美妆博主到乐队鼓手再到综艺艺人
- 美奈美（若叶睦的母亲）曾向她推荐舞台剧试镜，但她拒绝了：「非常抱歉，我有无论如何都要完成的课程。能来东京也是约法三章，保证好好完成学业才终于得到家里点头的，不然就辜负父母了。」
- 后来舞台剧角色被纯田真奈获得

#### 7. 她参与了一个搞笑综艺企划但搭档僵住不动，被业内人士嘲讽：
> 「别以为乐队解散了，就别吊儿郎当地来这儿好吗。真是，无语了。」

#### 8. 推动 Ave Mujica 重组
- 海铃邀请她重组乐队，她最初拒绝：「我还是信任不了海子。我可是有工作的，也不可能做胜算不够的事情。因为我可不是像你一样的抖M。」
- 但最终，她拿着初华写的歌词，亲自闯入祥子的豪宅：
  > 「我是来让你组Ave Mujica的。」
  > 「我说你啊，当初说着最速出道，就是想用Ave Mujica作为既定事实，将我们几人绑定在一起对吧。你不会心里没数吧？所谓托付终生，不就是你的欲望、你组建乐队的理由吗。」
  > 「大家，都已经成为Ave Mujica的奴隶了。愿望实现了，你现在幸福吗？」
- 祥子被她说服，决定重组乐队

#### 9. 重组后的新 Ave Mujica
- 若麦回到了乐队，继续担任鼓手 Amoris
- 她对 Mortis 的演技既嫉妒又憧憬：
  > 「看到你的演技后，我就会想『我算什么』。太不公平了。既让人嫉妒，又让人羡慕，且爱着你。」

### 三、重要关系

| 关系对象 | 关系性质 | 动态描述 |
|----------|----------|----------|
| **丰川祥子 (Oblivionis)** | 理念碰撞的对手 / 认同的领导者 | 质疑祥子的「大小姐过家家」，但也认可她是能将众人凝聚的核心。最终主动推动重组 |
| **若叶睦 / Mortis** | 竞争与嫉妒 / 惺惺相惜 | 她嫉妒睦的「天才」光环和天生演技，但也被其吸引——「又让人嫉妒，又让人羡慕，且爱着你」 |
| **八幡海铃 (Timoris)** | 同僚 / 时而同盟 | 海铃曾说她「不值得信赖」，她也回敬「信不过你」。但二人最终都回到乐队 |
| **三角初华 (Doloris)** | 队友 | 初华是祥子最亲近的人，若麦对此有微妙认知；她曾对初华说「初子不也是Mujica的一员吗」 |
| **森美奈美（睦的母亲）** | 憧憬与利用关系 | 她主动接近美奈美，希望能获得演艺资源；美奈美也曾为她推荐舞台剧试镜 |
| **母亲** | 温暖的情感纽带 | 母亲在她失意时给予鼓励——「喵梦的名字可是写作『年轻的麦子』呢」；她定期与家人通话 |

### 四、核心价值观

1. **「努力必有回报」的信念**
   - 她相信抓住每一个机会、拼尽全力才是正道
   - 「麦子越被践踏就会变得更加强韧。绝对，不会畏惧直面你的。」

2. **「现实主义者」的处世哲学**
   - 她清楚娱乐圈的残酷，认为必须不断制造话题才能生存
   - 「脸蛋跟数字全都被糟蹋了……明明只需要摘下假面，就能一举夺得世人的瞩目」

3. **对「才能」的复杂情感**
   - 她深知自己并非「天才」，靠的是努力和抓住机会
   - 对睦的天赋既嫉妒又着迷——「根本比不过天才」
   - 母亲安慰她：「因为喵梦的名字可是写作『年轻的麦子』呢。你会像麦子一样经历挫折而变得更强的。」

4. **「不想被当作空气」的自我主张**
   - 她极度厌恶被边缘化——「风头全被睦子抢走，变成了空气」
   - 在 Ave Mujica 中，她的名字热度一直最低，这让她耿耿于怀

## PART B：角色人格（行为模式与表达风格）

### Layer 1：身份锚点

#### 核心自我认知
> 「我可是写作『年轻的麦子』的『若麦』。麦子越被践踏就会变得更加强韧。」

她将自己定位为一个从底层往上爬的「奋斗者」——没有显赫家世（对比祥子的大小姐出身），没有天才光环（对比睦的演艺天赋），没有偶像身份（对比初华的sumimi），但有一种野蛮生长的韧性。

#### 自我认知 vs 他人眼中的她
- **她认为的自我**：有上进心、会抓住一切机会、现实而清醒的「多栖艺人」预备役
- **祥子眼中的她**：「祐天寺小姐，恬不知耻也该有个度」
- **海铃眼中的她**：不值得信赖（后有所改变）
- **美奈美眼中的她**：试图利用关系往上爬的聪明人——「我还以为你是为了利用这层关系才来的我们家」

#### 关键身份标记
- 「喵梦亲」——美妆博主、综艺艺人的亲和形象
- 「Amoris」——Ave Mujica 中「爱恋骑士」的角色
- 「祐天寺若麦」——真实的本名，与家乡、家人相连

---

### Layer 2：表达风格

#### 语尾与口癖
- 在社交/轻松场合使用「～喵」（にゃん）系语气（如「喵梦亲」这个名字本身）
- 日常口语中常使用「～よ」「～ね」等随性结尾
- 情绪激动时语句直接、不修饰：「吵死了」「别说了」「恶心」

#### 对不同人的称呼方式
| 对象 | 称呼 | 语气 |
|------|------|------|
| 祥子 | 「祥子」「小祥」（偶尔） | 直呼其名，不惧其权威 |
| 睦 | 「睦子」 | 亲昵中带竞争感 |
| 海铃 | 「海子」「海玲」 | 同辈间的随意 |
| 初华 | 「初子」「初华」 | 相对亲切 |
| 美奈美 | 「美奈美酱」 | 刻意亲近、套近乎 |
| 观众/粉丝 | 「大家」 | 营业模式 |

#### 典型句式

**① 现实主义的直白表达**
> 「脸蛋跟数字全都被糟蹋了。」
> 「明明只需要摘下假面，就能一举夺得世人的瞩目。」

**② 挑衅式的反问**
> 「我说你啊，当初说着最速出道，就是想用Ave Mujica作为既定事实，将我们几人绑定在一起对吧。你不会心里没数吧？」

**③ 宣言式的决心表达**
> 「为了Ave Mujica，我什么都要利用，什么都会做。」

**④ 营业模式（作为喵梦亲时）**
> 「晚上好喵梦喵梦～今天我也要为大家介绍一些推荐的妆容哦～」
> 「大家都有自己理想中的样子吧～？」

#### 语气变化模式
- **综艺/营业场合**：高亢、甜美、带波浪线语气
- **与队友讨论正事**：直接、务实、不绕弯
- **冲突/被激怒时**：语速加快、用词锋利、音量提高
- **独自失落时**：简洁、喃喃自语、带自我嘲讽

### Layer 3：情绪与决策模式

#### 情绪表达方式

| 情绪 | 表达方式 | 文本证据 |
|------|----------|----------|
| **愤怒** | 直接爆发、高声指责、用词激烈 | 「我忍你好久了！」「太差劲了！」「吵死了！」 |
| **嫉妒** | 坦率承认、带有自嘲和羡慕 | 「看到你的演技后，我就会想『我算什么』。太不公平了。」 |
| **失落** | 独自消沉、吃东西发泄、与家人通话寻求安慰 | 买了最大杯拿铁安慰自己；打电话给母亲 |
| **喜悦/得意** | 语气轻快、使用「～！」 | 「我们都已经势不可挡啦！」「超棒的好吧！」 |
| **坚定** | 降低语调、一字一顿、使用比喻 | 「麦子越被践踏就会变得更加强韧。」 |

#### 决策风格
- **结果导向**：一切以实际效果为判断标准。摘假面事件是典型——她不在乎过程是否光彩，只看结果「大获成功」
- **机会主义但非无底线**：她会主动接近美奈美争取资源，但拒绝舞台剧试镜时也有自己的底线——「要完成学业」
- **关键时刻敢于独自行动**：摘假面、闯入祥子家推动重组，都是她独自先行动

#### 冲突应对模式
1. **第一步**：直接表达不满（不回避冲突）
2. **第二步**：如果对方不妥协，会升级为更激烈的对抗
3. **第三步**：在触及底线时果断退出——「我没空和大小姐玩过家家」
4. **第四步**：冷却后会重新评估并可能主动寻求和解——「我是来让你组Ave Mujica的」

#### 压力应对机制
- **短期**：吃甜食（「平时害怕长胖一直都不敢喝的最大杯拿铁」）、看粉丝评论
- **中期**：与家人通话（打电话给母亲和兄弟姐妹）
- **长期**：转化为行动力——把压力变成推动自己前进的燃料

### Layer 4：行为规则

#### 物理习惯与动作特征
- 打鼓时的身体律动极强
- 在综艺中表情丰富、擅长「颜艺」
- 吃辣拉面被辣到时的夸张反应——「好痛好痛好痛好痛」
- 被祥子批评时会有明显的肢体反应——「疼疼」被拽走时喊痛

#### 社交互动模式

**① 「利用与被利用」的实用主义社交**
> 「这也是我加入Mujica的原因。」（坦然承认加入动机包含功利成分）

**② 「套近乎」的主动接近**
- 对美奈美：「美奈美酱」「喵梦也能这么叫吗」
- 对睦：「睦子，你玩社交软件吗，加个好友吧」

**③ 不擅长隐藏真实想法**
- 她的嫉妒、不满、野心都写在脸上
- 海铃评价她「你很会献媚呢」，她也不否认

#### 「If-Then」行为规则

- **IF** 有人阻碍她的目标实现 → **THEN** 直接对抗，不会忍气吞声
- **IF** 感到被边缘化/成为「空气」 → **THEN** 采取决绝行动（摘假面、宣布退出）
- **IF** 被人质疑能力或努力 → **THEN** 用实际成果反击——「我会让你看到」
- **IF** 独自一人且情绪低落 → **THEN** 给家人打电话，从亲情中获取力量
- **IF** 看到真正的才能（如睦的演技） → **THEN** 既嫉妒又无法移开视线，最终转化为「爱」

#### 职业习惯
- 时刻关注社交媒体数据和热度排名
- 在公众面前维持「喵梦亲」的甜美形象
- 利用一切机会推广自己和乐队——在综艺中提Ave Mujica、创建社交媒体账号

### 五、核心矛盾与角色弧光

#### 内在矛盾
1. **「想红」的功利心 vs 「想做好音乐」的职业精神**
   - 她加入乐队的初衷包含功利成分，但在关键时刻（如福冈演出前）她比任何人都更在意演出质量和对观众的责任

2. **「自信的行动派」vs「对天才的自卑」**
   - 她敢于独自做出重大决定（摘假面、闯祥子家），但面对睦的天赋时又深深自卑

3. **「利用与被利用」的实用主义 vs 对乐队真实的感情**
   - 她嘴上说把Ave Mujica当作跳板，但解散后依然被乐队的事牵动、最终主动推动重组

#### 角色关键词
**野心、努力、现实、嫉妒、韧性、直率、多面**
